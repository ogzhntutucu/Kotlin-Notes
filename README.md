## 🇬🇧 English

This repository holds the notes I collect while learning Kotlin. It is not tied to a single course — over time I keep adding everything I learn from different courses, videos, and docs. Think of it as a living archive of "everything I've learned about Kotlin."

I write the notes in [Obsidian](https://obsidian.md), so this repo is also an Obsidian vault. Its main purpose is personal reference, but it also serves as a place to quickly show my Kotlin background if needed — and it's public, so anyone is welcome to benefit from it.

> **Note:** The notes are written mostly in Turkish, with plenty of Kotlin code examples throughout.
